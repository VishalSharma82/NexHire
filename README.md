# AI Interview Coach - Elite Edition 🏆

This is the upgraded "Elite" version of the AI Interview Coach, featuring OpenAI intelligence, live coding practice, and voice interaction.

## Elite Features
- **Real-time AI Mentorship**: Powered by OpenAI GPT-4o for professional, context-aware coaching.
- **Interactive Coding Sandbox**: Practice technical problems in a built-in code editor.
- **Voice-to-Text Practice**: Use the Web Speech API to answer mock interview questions out loud.
- **Resumé Intelligence**: Paste your experience for a tailored interview session.
- **Professional Interview Simulator**: Integrated timer, persona selection, and AI performance scoring.
- **Learning Roadmap**: Structured tracks for Frontend, Backend, and System Design.

## Getting Started

1. **Clone and Install**:
   ```bash
   npm install
   ```

2. **Configure API Key**:
   Create a `.env` file or enter your key directly in the app's settings.
   ```env
   VITE_OPENAI_API_KEY=your_key_here
   ```

3. **Run Dev Server**:
   ```bash
   npm run dev
   ```

## Tech Stack
- React 19 + Vite
- Tailwind CSS v4
- Framer Motion
- Lucide React
- OpenAI API
- PrismJS (Code Highlighting)
- Web Speech API (Voice Input)
